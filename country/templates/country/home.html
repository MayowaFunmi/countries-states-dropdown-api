devMayor

This is a 3-in-1 project:

  1. A dependent dropdown list of world countries(ISO 3166) and their respective states/provinces/subdivisions,
  2. A dependent dropdown list of Nigerian states and their respective local government ares
  3. A restful api (using django rest framework) providing data for the two listed above and can be consumed by any javascript frontend library/framework

Features

  • HTML, CSS, JavaScript, Ajax, Python, Django, DjangoRestFramework
  • List World Countries
  • Dependent/Dropdown of world countries and states/provinces
  • Dependent/Dropdown of Nigerian states and local government areas
  • Django Restful api to provide data
  • No css (except for this page)
